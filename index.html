סנכרון קטלוג הספרים לענן

חיבור האתר ל-Supabase יאפשר לך לגשת לקטלוג הספרים שלך מכל מכשיר (מחשב, טאבלט, טלפון). הנתונים יישמרו ויסתנכרנו בזמן אמת.

איך משיגים את המפתחות? 1. פתחי פרויקט חינמי ב-supabase.com.
2. צרי טבלה בשם books.
3. הוסיפי עמודות: id (טקסט, מפתח ראשי), title (טקסט), author (טקסט), rating (מספר שלם, מאופשר כריק).
4. בטלי את ה-RLS (או הגדירי גישה חופשית ל-anon) בהגדרות הטבלה.
5. העתיקי את ה-Project URL וה-Anon Key מהגדרות ה-API והדביקי כאן.
